# Báo Cáo Tóm Tắt: "GenAI – Hành trình từ AI Engineer truyền thống đến GenAI Engineer thực chiến"


## 1. Mục tiêu tham dự
- Nắm bắt xu hướng ứng dụng Generative AI (GenAI) trong doanh nghiệp 2025–2026
- Hiểu sự khác biệt giữa AI Engineer truyền thống và GenAI Engineer
- Xây dựng lộ trình phát triển kỹ năng phù hợp với nhu cầu thị trường

## 2. Nội dung chính

### 2.1. So sánh AI Engineer truyền thống và GenAI Engineer

| Tiêu chí | AI Engineer truyền thống | GenAI Engineer (2025) |
|----------|-------------------------|----------------------|
| **Mục tiêu công việc** | Xây dựng mô hình ML từ đầu | Tích hợp GenAI vào ứng dụng doanh nghiệp |
| **Kỹ năng cốt lõi** | Toán, Thống kê, Python, R, ML algorithms | Prompt Engineering, Fine-tuning, Software Engineering |
| **Công cụ chính** | TensorFlow, PyTorch, Spark | API LLM (GPT, Claude), LangChain, Vector DB |
| **Ứng dụng thực tế** | Nghiên cứu dài hạn | Giải pháp nhanh: chatbot, kiểm kho, tự động hóa |

**Nhận xét:** GenAI Engineer đòi hỏi kỹ năng Software Engineering mạnh (70%) kết hợp kiến thức AI (30%), tập trung vào triển khai production với chi phí thấp, latency tối ưu và bảo mật dữ liệu.

### 2.2. Vai trò GenAI trong IT
- **Software Developer:** Tăng tốc coding 30–50% với AI coding assistants
- **Cloud Engineer:** Tự động debug, phân tích log, viết Infrastructure-as-Code
- **Doanh nghiệp:** Ứng dụng GenAI cho bài toán thực tế như kiểm tra kệ hàng, chatbot nội bộ, tổng hợp báo cáo

### 2.3. Những hiểu lầm phổ biến về GenAI
- **Prompt Engineering đơn giản:** Thực tế cần hiểu Context Window, Chain-of-Thought, Few-shot để tạo output ổn định
- **GenAI thay thế coder:** Sai, coder biết dùng GenAI sẽ thay thế coder không biết dùng
- **Chỉ cần gọi API là đủ:** Production yêu cầu tối ưu latency, chi phí và bảo mật (RAG, caching, quantization)

### 2.4. Giới thiệu First Cloud Journey
- **Thành lập:** 2020 (5 năm)
- **Thành tựu:** Đào tạo >5.000 học viên, hỗ trợ ~200 bạn tìm việc (chủ yếu Cloud & GenAI)
- **Sứ mệnh:** Phát triển lãnh đạo kỹ thuật Việt Nam, đóng góp cho cộng đồng

## 3. Giá trị nhận được
- Hiểu rõ GenAI là yêu cầu bắt buộc cho dự án doanh nghiệp 2025–2026
- Xác định kỹ năng then chốt: Software Engineering + Prompt Engineering + Production optimization
- Nhận lộ trình học tập từ First Cloud Journey để trở thành GenAI Engineer thực chiến trong 6–12 tháng

## 4. Kế hoạch hành động (Q1–Q2/2026)
- Nâng cao kỹ năng backend (FastAPI, Docker, PostgreSQL)
- Thành thạo Prompt Engineering và Retrieval-Augmented Generation (RAG)
- Xây dựng 1–2 dự án GenAI thực tế (chatbot nội bộ hoặc tự động hóa quy trình)
- Tìm hiểu Fine-tuning (LoRA/QLoRA) và tối ưu production (vLLM, caching)

## 5. Đề xuất cho công ty
- Xem xét tích hợp GenAI vào quy trình nội bộ (VD: chatbot hỗ trợ nhân sự, tự động hóa báo cáo)
- Đào tạo đội ngũ IT về Prompt Engineering và RAG để tăng hiệu suất
- Khai thác dữ liệu nội bộ để xây dựng giải pháp GenAI tùy chỉnh, đảm bảo bảo mật
